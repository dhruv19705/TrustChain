# 📊 Dataset Documentation

## Dataset Name
Credit Card Fraud Detection Dataset

## Source
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## Description

The dataset contains anonymized credit card transactions made by European cardholders in September 2013.

Total Records: 284,807  
Fraud Cases: 492  
Fraud Rate: 0.172%

The dataset is highly imbalanced.

---

## Features

Time – Seconds elapsed between transactions  
V1–V28 – PCA transformed numerical features  
Amount – Transaction amount  
Class – Target variable (0 = Legitimate, 1 = Fraud)

---

## Preprocessing Used

- Feature scaling
- Train-test split
- Class imbalance handling
- Model training and evaluation

---

## Suitability

This dataset is widely used for academic fraud detection research and is appropriate for:

- Supervised classification
- Anomaly detection
- Imbalanced learning experiments
- Blockchain-integrated validation simulation
